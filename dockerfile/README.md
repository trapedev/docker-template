# About Dockerfile

## 仕様
基本命令は以下の通り．

|  命令  |  効果  |
| ---- | ---- |
|  `FROM`  |  ベースイメージの指定  |
|  `RUN`  |  任意のコマンドの実行  |
|  `COPY`  |  ホストマシンのファイルをイメージに追加  |
|  `CMD`  |  デフォルト命令の指定  |
